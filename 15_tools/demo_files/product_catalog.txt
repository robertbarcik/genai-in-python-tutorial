Category: Identity and Access Management
Version: 3.0.0
Release Date: January 2025

DESCRIPTION:
SecureAuth 360 is a comprehensive identity and access management solution that
provides single sign-on (SSO), multi-factor authentication (MFA), and user
lifecycle management for organizations of any size.

KEY FEATURES:
- Single Sign-On (SSO) with support for SAML 2.0 and OAuth 2.0/OIDC
- Multi-factor authentication (authenticator apps, SMS, email, hardware keys)
- Passwordless authentication options (WebAuthn, magic links)
- User provisioning and deprovisioning automation
- Directory integration (Active Directory, LDAP, Azure AD)
- Self-service password reset portal
- Detailed security analytics and threat detection
- Compliance reporting (SOC 2, GDPR, HIPAA)

TECHNICAL REQUIREMENTS:
- Cloud-hosted (SaaS) - no infrastructure required
- On-premise option available for Enterprise customers
- Browser support: Chrome, Firefox, Safari, Edge (latest 2 versions)
- Mobile apps: iOS 14+, Android 10+

PRICING:
- Basic: $3/user/month (SSO only, minimum 50 users)
- Standard: $6/user/month (SSO + MFA, minimum 50 users)
- Premium: $12/user/month (Full feature set, minimum 25 users)
- Enterprise: Custom pricing (on-premise option, advanced features)

COMPLIANCE CERTIFICATIONS:
- SOC 2 Type II
- ISO 27001
- GDPR compliant
- HIPAA compliant (Business Associate Agreement available)
